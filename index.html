about this site...

the page turns

I had fun building this. And I could talk a lot about this effect. But to be brief, my goal was to create a page flip that had html content. Not pictures disguised as words. The part of the page under the rings needed to animate under the rings. Likewise the rest of the page needed to animate over the rings. The pages also are alligned a hair off to create depth. The depth is different on each side, so the animation needs to adjust vertically. The animations need to allow the user to be able to click on any tab to get to any page. Among other nitpicky requirements, it needs to be updated easly. The end result is it works pretty seamlessly using both CSS and jQuery.

distressed buttons

The navigation buttons are a fun little effect. I am again using CSS and jQuery. The effect is tied into the page slides. The rollover animation is using mulitple backgrounds. The second background shrinks into place by adjuesting the background size. The higlight effect is using yet another background image. The jQuery toggles the class and CSS transitions the background size.

the paperstack

The page slides are all jQuery. The HTML is all stacked into divs. The jQuery animates the top page out of frame while revealing and reseting the next page underneath.

my skill set...

I love coding and enjoy coding games in my spare time, or when projects allow. I feel that games often push the envelope and drive inovation. Video games are what pushed computers from simple number cruchers to massive multimedia machines. Well, games are at it again pushing the web forward (mobile is pushing it ahaed to). Being in a field that is constantly growing and improving it is imperative to keep learning and luckily its a joy to. While I am a huge fan of blending many scripts and techonogies into a website, each has their role in building the overall experience.

html semantics

HTML5 brings us context to the content we are placing on the web. This is helpful for search engine optimization and the almighty Google ranking. HTML is the bones of the site and needs to be well structured for a positive user experience and flow of information.

css styling

Add some flare and personality to the site. CSS is more than pure aesthetics and making lists pretty. CSS also drives the user experience by adding a visual heirarchy and order to the content. CSS is progressing so far that Flash is becoming irrelavent.

jquery

JavaScript and jQuery is used to add functionality to a site. There are many dynamic and responsive functions JavaScript can do to add to the user experience. Javascript is a joy to work with because it does so much. It can be used to simply agument an effect like a few on this site. JavaScript is also capable of Object Oriented Programming and can be the framework for some powerful applications, or fun games.

php

Creating a dynamic framwork for a site to operate on is where the gold is in web. PHP operating behind the sences and talking to databases to call up content for the webpage is the heart of most websites. Most all Content Management Systems operate with PHP as a base.

mysql

Databases are fun.(Really?) I just enjoy making things work, and MySQL (and SQL) is where the data can be stored to get work done.

adobe cs

There are many tools available to design and develop websites. Adobe Creative Suite has some of the best tools in Photoshop and Illustrator. I find it easy to loose hours designing and tweaking mocks. Building a mock is a crutial step in the web development process as it could save hours of headache when coding the website.
